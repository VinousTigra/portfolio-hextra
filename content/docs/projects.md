# Проекты

## Сайт-портфолио на Hugo

Статический сайт, созданный с использованием Hugo и темы Hextra.

В проекте реализованы:

- локальная сборка сайта;
- страницы портфолио;
- публикация через GitHub Pages;
- подготовка к автоматическому деплою через GitHub Actions.

## Учебные frontend-задания

В рамках курса планируется добавить:

- адаптивную верстку;
- Bootstrap-компоненты;
- JavaScript-интерактивность;
- Vue-компоненты.
